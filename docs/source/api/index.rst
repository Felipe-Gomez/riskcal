API Reference
-------------

The library provides three main modules:

* **riskcal.analysis** - Computing privacy risk measures from different privacy representations
* **riskcal.calibration** - Calibrating noise parameters to achieve target risk guarantees
* **riskcal.accountants** - Privacy accountants which support risk measurement in their API
